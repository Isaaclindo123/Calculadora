# Bem vindo a minha calculadora em C#
Essa é a minha mais nova calculadora em c#, não é muito bonite, mas é funcional e eficiente
# O que tem na calculadora?
As coisas básicas que tem, como:
- Um botão de +
- Um botão de -
- Um botão para dividir /
- Um botão para multiplicar *
- Um botão para limpar a tela AC (All Clear)
- Um botão de porcentagem %
- Um botão igual =
- Um botão de vírgula
- Os botões de 0 à 9
- E um histórico funcional
# E isso é tudo (:
Obrigado por usar
Feito no Visual Studio com Windows Forms
